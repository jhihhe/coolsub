**coolsub** is a premium subtitle styling tool designed for users who seek the ultimate cinematic experience. It transforms common subtitle files (SRT, VTT, ASS, etc.) into ASS subtitles with high-end typography and cinematic visual styles.

## ✨ Features

- **Immersive Visuals**: Built-in meticulously designed styles, including "Immersive Serif", "Modern Minimalist", and the iconic "Netflix Orange".
- **Encoding Auto-detection**: Smartly identifies and fixes various encodings like UTF-8, UTF-16🔍搜 UTF-16, and files with BOM, eliminating garbled text.
- **Real-time Preview**: Instantly preview how different styles look before processing.
- **Minimalist Interaction**: Supports batch processing via drag-and-drop, generating styled files directly in the source folder.
- **macOS Native Feel**: A modern GUI built with PySide6, deeply optimized for macOS.

## 🎨 Built-in Styles

- 📜 **Immersive Serif**: Extremely restrained, cinematic texture like reading a paper book.
- 📱 **Modern Minimalist**: Clean and crisp, perfect for modern series.
- 🎬 **Classic Cinematic**: Nostalgic and retro, with a strong filmic atmosphere.
- 🍊 **Netflix Orange**: Iconic Netflix orange color with high recognition.
- 🎥 **Documentary Yellow**: Professional and eye-catching documentary style.

## 🚀 Quick Start

### Install Dependencies
```bash
pip install PySide6 pysubs2
```

### Run Application
```bash
python3 src/main.py
```

### Build as .app
```bash
python3 build.py
```
